## Goal

## Work

- [ ] Scan databases, tables, engines, columns, types, defaults, primary keys, unique keys, foreign keys, indexes, generated columns, views, and triggers.
- [ ] Calculate schema fingerprint and monotonic local epoch.
- [ ] Store scan timestamp, source, completeness, and errors.
- [ ] Detect unsupported and non-transactional table engines.
- [ ] Add periodic and manual rescan.
- [ ] Expose catalog through API and GUI.

## Tests

- [ ] Catalog matches fixture metadata.
- [ ] Add, alter, rename, and drop change fingerprint.
- [ ] Partial scan never marks schema safe.

## Done when

- [ ] Catalog survives restart through versioned local metadata.
- [ ] Evidence recorded and status set to `DONE`.
