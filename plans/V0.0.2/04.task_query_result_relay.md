## Evidence

Commit/PR:

Corpus: `TestIntegrationTransparentRelayMatchesDirectDriver` and `TestIntegrationPooledGatewayTransactionsAndFanIn` pass against MariaDB 11.7.2 and Oracle MySQL 8.4.10. The transparent compatibility path forwards raw column-origin fields and all other packets byte-for-byte. Its native-driver differential covers multi-results, null, binary, Unicode, decimal, a 4 MiB result, error code/SQLSTATE, affected rows, and insert ID. The pooled path covers driver-visible column name/type/nullability/precision/scale, warning counts on the same physical connection, a 48 MiB result under a 40 MiB heap-growth ceiling, rollback, commit, and 64-client fan-in. A differential test found and fixed incorrect DECIMAL precision reconstruction. Unsupported pooled multi-statement negotiation fails during handshake.
