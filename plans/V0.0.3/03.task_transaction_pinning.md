## Goal

Preserve one upstream transaction exactly as one database transaction.

## Work

- [ ] Detect explicit and implicit transaction start.
- [ ] Pin before first transaction-sensitive operation.
- [ ] Keep all statements, savepoints, commit, and rollback on same upstream connection.
- [ ] Return commit success only after upstream success.
- [ ] On disconnect, close upstream connection so MySQL rolls back uncommitted work.
- [ ] On lost commit response, return connection failure and never retry.
- [ ] Invalidate affected cache generations before releasing successful commit response hook.

## Tests

- [ ] Commit, rollback, savepoint, deadlock, lock timeout, and client disconnect tests.
- [ ] Kill accelerator before commit, during commit, and after upstream commit.
- [ ] Assert no partial transaction state.

## Done when

- [ ] Guarantee A in atomicity contract is covered by named tests.
- [ ] No transaction can enter shared cache or switch upstream connection.
- [ ] Evidence recorded and status set to `DONE`.
