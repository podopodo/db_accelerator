Status: `DONE`

## Goal

Preserve one upstream transaction exactly as one database transaction.

## Work

- [x] Detect explicit and implicit transaction start.
- [x] Pin before first transaction-sensitive operation.
- [x] Keep all statements, savepoints, commit, and rollback on same upstream connection.
- [x] Return commit success only after upstream success.
- [x] On disconnect, close upstream connection so MySQL rolls back uncommitted work.
- [x] On lost commit response, return connection failure and never retry.
- [x] Invalidate affected cache generations before releasing successful commit response hook.

## Tests

- [x] Commit, rollback, savepoint, deadlock, lock timeout, and client disconnect tests.
- [x] Kill accelerator before commit, during commit, and after upstream commit.
- [x] Assert no partial transaction state.

## Done when
